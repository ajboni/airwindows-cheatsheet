# Airwindows Cheatsheet

Cheatsheet for Airwindows plugins.
This project aims to build an automated frontend for the airwindowpedia documentation file.

# Features

- Category filters
- Search filters
- Favorites
- Automatic Builds.
- Mobile friendly.
